# Source Header Templates

These templates are intended for OpenCredential source files while preserving
the original BSD-3-Clause licensing obligations inherited from pGina.

## 1. New OpenCredential file

Use this when the file is newly created for OpenCredential:

```text
// OpenCredential - Source code
// https://github.com/pedropablobm/OpenCredential
//
// Copyright (c) 2024, Pedro Bermudez
// All rights reserved.
//
// This file is part of OpenCredential, an unofficial fork of pGina.
// It is distributed under the BSD-3-Clause license terms used by the
// original pGina project unless stated otherwise.
//
// For more details, see the LICENSE and NOTICE files included with
// this distribution.
```

## 2. File based on original pGina source code

Use this when a file comes from pGina and has been substantially modified:

```text
// OpenCredential - Source code
// https://github.com/pedropablobm/OpenCredential
//
// Based on original pGina source code.
// Further modifications Copyright (c) 2024, Pedro Bermudez.
//
// This file is part of OpenCredential, an unofficial fork of pGina.
// It remains subject to the BSD-3-Clause license terms of the original
// pGina project.
//
// For more details, see the LICENSE and NOTICE files included with
// this distribution.
```

## Notes

- Do not remove original copyright notices from inherited pGina files.
- Do not rewrite the original BSD-3-Clause text in a way that changes its
  meaning.
- Do not present OpenCredential as endorsed by the pGina Team.
